Página 4 - Códigos-Fontes e afins

kbd O elemento HTML kbd é usado para definir a entrada do teclado. O conteúdo interno é exibido na fonte monoespaçada padrão do navegador.
samp O elemento HTML samp é usado para definir a saída de amostra de um programa de computador. O conteúdo interno é exibido na fonte monoespaçada padrão do navegador.
code O elemento HTML code é usado para definir um trecho de código de computador. O conteúdo interno é exibido na fonte monoespaçada padrão do navegador.
var O elemento HTML var é usado para definir uma variável em programação ou em uma expressão matemática. O conteúdo interno normalmente é exibido em itálico.
pre A tag pre elemento define texto pré-formatado
