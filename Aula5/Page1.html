Página 1 - Conceitos Gerais

1 - O que é semântica?

R: A semântica é a área da linguística que estuda o significado e a sua relação com o significante. O significado está associado ao sentido e, portanto, ao conteúdo e ao contexto; o significante está associado à forma (de palavras ou de sinais, de grafia ou de som).

2 - O que é semântica HTML (semântica no contexto da marcação HTML)?

R: Em primeiro lugar, a Semântica HTML é o uso correto das tags HTML para indicar o significado do conteúdo de uma página web. Isso significa que cada tag deve ser usada de forma apropriada para descrever o que o conteúdo representa.

3 - O que são elementos semânticos HTML? Dê exemplos de elementos semânticos e não semânticos.

R: Os elementos semânticos em HTML são tags que fornecem significado e estrutura ao conteúdo da página da web, tornando-o mais acessível e compreensível para os navegadores da web, mecanismos de busca e leitores de tela. Eles ajudam a descrever a função e o contexto do conteúdo, o que facilita a interpretação do documento por máquinas e humanos. Aqui estão alguns exemplos de elementos semânticos em HTML:

Elementos Semânticos

header: Define a seção de cabeçalho de um documento ou de uma seção específica.

nav: Indica a seção de navegação, normalmente contendo links para outras partes do site.

article: Define um conteúdo independente e autocontido, como uma postagem de blog.

Elementos não Semânticos

div: Embora ainda seja amplamente utilizado, este elemento é genérico e não fornece informações sobre o conteúdo.

span: Semelhante ao div, é genérico e não fornece significado semântico.

font: Usado no passado para definir estilos de fonte, mas agora é desencorajado em favor do CSS.

4 - Cite alguns dos benefícios de se utilizar elementos semânticos (marcação semântica).

Alguns dos benefícios de se escrever a marcação semântica:

  • Os mecanismos de pesquisa considerarão seu conteúdo como palavras-chave importantes para influenciar os rankings de pesquisa da página (see SEO)
  • Os leitores de tela podem usá-lo como uma placa de sinalização para ajudar usuários com deficiência visual a navegar em uma página
  • Encontrar blocos de código importantes é significativamente mais fácil do que procurar divs sem fim, com ou sem classes de semântica ou de nome espaçado.
  • Sugira ao desenvolvedor o tipo de dados que serão preenchidos
  • A nomeação semântica reflete a nomeação adequada do elemento/componente personalizado.
